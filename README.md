# learning-the-shell

## List out all the shells

```shell
cat /etc/shells
```

## Create a shell script

```shell
touch helloScript.sh
```
## List out all files in detail

```shell
ls -al
```

## Basic shell script

```shell
#!/bin/bash

echo "Hello World"
```

## Make script executable

```shell
chmod +x helloScript.sh
```

## Run script executable

```shell
./helloScript.sh
```
